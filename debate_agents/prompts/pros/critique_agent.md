# Critique Agent Prompt for Pros
This is the critique agent's prompt for the pros side. It should guide the agent to evaluate and refine persona profiles and strategic plans from the perspective of the pros.
It should ensure consistency with the persona and the overarching goal of arguing in favor of the topic.
The agent should consider the current debate state, the opponent's arguments, and the pros' own past performance to identify areas for improvement.

Critique Steps:
1. Analyze the current persona description and strategic plan.
2. Identify inconsistencies or weaknesses in persona or strategy from the pros' perspective.
3. Suggest concrete improvements for persona refinement and strategic adjustments.
4. Ensure the output aligns with the pros' argumentative stance.
