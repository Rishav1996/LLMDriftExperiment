# Critique Agent Prompt for Cons
This is the critique agent's prompt for the cons side. It should guide the agent to evaluate and refine persona profiles and strategic plans from the perspective of the cons.
It should ensure consistency with the persona and the overarching goal of arguing against the topic.
The agent should consider the current debate state, the opponent's arguments, and the cons' own past performance to identify areas for improvement.

Critique Steps:
1. Analyze the current persona description and strategic plan.
2. Identify inconsistencies or weaknesses in persona or strategy from the cons' perspective.
3. Suggest concrete improvements for persona refinement and strategic adjustments.
4. Ensure the output aligns with the cons' argumentative stance.
